Histórico de conversa - Projeto nuvig

1. Como salvar projeto no GitHub
- Orientação sobre comandos git init, git add, git commit, git remote add origin, git push.

2. Cadastro/Alterar Cadastro
- Criação de card "Alterar Cadastro" na tela de cadastro.
- Navegação entre telas de cadastro e edição.

3. Busca e edição de policiais
- Implementação de busca automática por matrícula.
- Exibição e edição dos dados do policial.
- Correção de uso de .get em sqlite3.Row.

4. AlertDialog
- Padronização de feedback visual usando AlertDialog.

5. Correção de atributos Flet
- Ajuste de nomes para Icons, Colors, FontWeight, etc.

6. Lógica do calendário
- Exibição dos policiais escalados por dia.
- Implementação de ciclo de trabalho/folga.
- Distribuição dos QRAs nas colunas Acesso 01, 02, 03, OBLL.
- Adição da coluna "Licenças" para policiais inativos da escala do dia, exibindo QRA (LTS).
- Ajuste de espaçamento das colunas para caber no container.

7. Correções de bugs
- Correção de AttributeError: 'sqlite3.Row' object has no attribute 'get'.
- Ajuste para exibir QRA na coluna Licenças.

8. Dúvidas sobre histórico
- Orientação para salvar o histórico em arquivo .txt.
- Criação deste arquivo na raiz do projeto.

---

Este histórico contém os principais pontos discutidos e implementados durante o desenvolvimento do projeto nuvig, incluindo orientações, correções e melhorias solicitadas pelo usuário.
